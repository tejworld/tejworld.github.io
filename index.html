About Us
Project background, model details & team
Project Introduction
CrowdSense is an intelligent crowd management system built to address the growing safety challenges at large-scale public events — concerts, sports matches, religious gatherings, and transit hubs.

Using computer vision and deep learning, CrowdSense enables security teams and operators to quantify crowd density and count individuals in real-time from live CCTV feeds, uploaded footage, or static images — without manual headcounts or expensive on-ground personnel.

The system integrates two independently-trained AI models through a lightweight Flask backend, surfaced via this desktop-optimised browser interface deployable on GitHub Pages. Alerts are generated automatically based on configurable density thresholds, giving operators actionable intelligence before situations escalate.
Model 1 — Crowd Density Estimation

What it does: Classifies crowd density from an image or video frame into four levels: Low Medium High Critical

Architecture: CNN-based regression model trained on annotated crowd images. Outputs a continuous density score (0.00–1.00) mapped to discrete alert levels via threshold rules.

Dataset: ShanghaiTech Crowd Counting Dataset (Part A & B), augmented with synthetically-generated dense and sparse scenes.

Evaluation:

MAE Mean Absolute Error on density map reconstruction
MSE Mean Squared Error per frame

Known Limitations: Performance degrades on highly occluded scenes and low-resolution inputs (<320px).

Model 2 — People Counting

What it does: Returns the exact number of people detected in a given frame using bounding-box detection.

Architecture: YOLOv8-based object detector fine-tuned on pedestrian datasets. Outputs bounding-box coordinates and a total person count per frame.

Dataset: CrowdHuman dataset combined with custom annotated frames from Indian public-gathering contexts.

Evaluation:

nAP Normalised Average Precision on pedestrian class
Recall Detection recall at IoU threshold 0.5

Known Limitations: Counts may underestimate in heavily overlapping crowds; accuracy drops in night-time or low-light scenes.
